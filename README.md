# New Vision Veterinary Hospital

An appointment booking and management system for veterinary clinics, built with the latest web technologies.

## Description

This project provides a comprehensive solution for veterinary clinic management. It includes a public-facing landing page with an appointment booking form and a full-featured admin dashboard for managing clinics, availability, and bookings.

The system allows administrators to confirm or reject bookings and automatically notifies clients via email.

## Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org) (App Router, Server Actions)
- **Authentication:** [Next-Auth (v5)](https://next-auth.js.org/)
- **Database:** [PostgreSQL (Neon)](https://neon.tech/) with [Drizzle ORM](https://orm.drizzle.team)
- **Email:** [React Email](https://react.email) and [Resend](https://resend.com)
- **UI:** [Tailwind CSS](https://tailwindcss.com) and [shadcn/ui](https://ui.shadcn.com)

## Key Features

- [x] Full Localization (English)
- [x] Responsive Landing Page with Booking Form
- [x] Admin Dashboard for Clinic Management
- [x] Automated Email Notifications (Booking states, Confirmations)
- [x] User Authentication and Profile Management
- [x] Real-time Availability Management
- [x] Advanced Booking Filtering and Management

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run the development server: `npm run dev`
